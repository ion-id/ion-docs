# ion-docs
This repository contains documents relevant to the entire Indonesia Open Network. Draft documents will be indicated and relying on them will carry a risk. 
The structure of the folders are:
- `drafts` - Contains draft version of documents produced by ION team.
- `released` - Contains release version of documents produced by ION team.
- `general` - These are informational notes, important links and other stuff that do not strictly fall under publishing process.
