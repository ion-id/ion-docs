# TradeResource

`TradeResource` describes catalog resource attributes for trade use cases.

The schema is closed: only `category`, `identity`, and `extraAttributes` are allowed. The `extraAttributes` field delegates NP-specific JSON-LD attributes to the local `IONExtraAttributes` schema.

## Scope

- Container: `resourceAttributes`
- Protocol version: `2.0`
- Semantic model: `generalised`
- Namespace prefix: `rcra`
- Namespace: `https://schema.beckn.io/TradeResource#`
